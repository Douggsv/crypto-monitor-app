# 📈 Crypto Monitor App

A modern cryptocurrency monitoring dashboard that displays real-time prices of the top 12 cryptocurrencies using the Binance API.

Built with a clean and responsive interface, this application focuses on real-time data visualization and performance using modern web technologies.

---

## 🚀 Features

* 📊 Real-time cryptocurrency price tracking
* 🔝 Displays top 12 cryptocurrencies
* ⚡ Fast and optimized UI with Next.js
* 🎨 Clean and responsive design with Tailwind CSS
* 🔗 Live data integration using Binance API
* 🧠 Built with TypeScript for better scalability

---

## 🛠️ Technologies Used

* **Next.js** – React framework for production
* **TypeScript** – Type-safe JavaScript
* **Tailwind CSS** – Utility-first styling
* **Binance API** – Real-time crypto market data

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Douggsv/crypto-monitor-app.git
```

2. Navigate into the project:

```bash
cd crypto-monitor-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open your browser at:

```bash
http://localhost:3000
```
